# End-to-End-E-Commerce-Analytics-Dashboard
 [E-Commerce Analytics Dashboard]
 1. Short Description

An end-to-end data analytics project that transforms raw e-commerce data into actionable business insights. The project includes data scraping, data cleaning using Python (Pandas), feature engineering, and the development of a multi-page interactive Power BI dashboard for product, customer, pricing, and category analysis.

2.Tech Stack
 Python
 Pandas
 Web Scraping
 Power BI
 DAX
 CSV
 
3. Data Source
E-commerce product data collected through web scraping
Dataset includes:
Product Name
Category
Price
Rating Score
Rating Count
Popularity Metrics

4. Features
Data Collection
Scraped raw e-commerce product data from online sources

Renamed and standardized columns

Created Price Categories

Prepared data for dashboard analysis
